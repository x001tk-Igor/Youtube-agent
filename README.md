# 🤖 YouTube AI Agent

Локальный ИИ-агент для анализа видео и каналов YouTube через Ollama.

## ✨ Возможности

- 📹 Саммари видео по вашей задаче
- 🔍 Поиск информации по всему каналу
- 💾 Сохранение результатов (txt, md, pdf)
- 🤖 Работает полностью локально (без API ключей)
